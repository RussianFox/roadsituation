# roadsituation
Login on your server with centOS7 as **root**

Create directory **/git** and clone repository
```
yum install git -y
mkdir /git
cd /git
git clone https://github.com/RussianFox/roadsituation.git
```

Run install and setup
```
cd /git/roadsituation/server
sh ./install.sh
sh ./setup.install
```
