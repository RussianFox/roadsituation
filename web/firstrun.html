Как этим пользоваться

Приблизте карту к нужному месту, обьекты подгрузятся автоматически

Чтобы добавить свой обьект, дважды тапните по карте

В углу есть индикатор загрузки

Также в углу есть фильтр, где можно выбрать типы обьектов, а также карту подложки

Если индикатор загрузки пропал, а обьектов не появилось, значит в этой области ничего нет, из того, что выбрано в фильтре

Если вместо индикатора загрузки знак СТОП, то просто следует увеличить масштаб

Об этом проекте

Этот проект является попыткой создать единый, свободный, открытый, общедоступный сервис дорожных событий, чтобы любое приложение или любой человек мог им воспользоваться

На данный момент я экспериментирую с форматами данных и разрабатываю API
